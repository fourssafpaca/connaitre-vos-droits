# FO URSSAF PACA — Infographies & Fiches Mémo

Site d'information syndicale réalisé par **Force Ouvrière URSSAF PACA**.

## 🚀 Déploiement sur GitHub Pages

1. Créez un compte sur [github.com](https://github.com) (gratuit)
2. Cliquez **New repository** → nommez-le `fo-urssaf-paca` → **Create repository**
3. Cliquez **"uploading an existing file"**
4. Glissez-déposez **tout le contenu** de ce dossier (index.html + dossiers infographies/ et fiches-memo/)
5. Cliquez **Commit changes**
6. Allez dans **Settings** → **Pages** → Source : **Deploy from a branch** → Branche `main`, dossier `/ (root)` → **Save**
7. Patientez 1-2 minutes, votre site sera accessible à : `https://VOTRE-USERNAME.github.io/fo-urssaf-paca/`

## 📄 Ajouter un nouveau document

1. Déposez l'infographie dans `infographies/` et la fiche mémo dans `fiches-memo/`
2. Mettez à jour `index.html` en ajoutant une nouvelle `doc-card`
3. Commitez — le site se met à jour automatiquement
